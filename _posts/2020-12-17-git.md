要注意 Github 对 Repository 是有大小限制的。

> We recommend repositories remain small, **ideally less than 1 GB**, and less than 5 GB is strongly recommended. 

<https://docs.github.com/en/free-pro-team@latest/github/managing-large-files/what-is-my-disk-quota>

一旦大了会出现各种奇怪的错误。

所以这就是为什么 Github 不能做私人网盘。
