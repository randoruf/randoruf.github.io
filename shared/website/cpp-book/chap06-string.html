String

It is recommended to use <string> in C++ library.

Read https://docs.microsoft.com/en-us/cpp/standard-library/string?view=msvc-160

C-Style String

You should note that 

ca1  == ca2

will just compare the pointers instead of comparing the whole string.

How string in C is created is not interested here, refere to C++ Primer pp122.

String to C-Style String with c_str() member method. 

char *str = std::string("Hello World").c_str();

C-style string is array 

const char ca1[] = "A string example"; 
const char ca2[] = "A different string"; 
if (ca1 < ca2) // error: unrelated addresses.
