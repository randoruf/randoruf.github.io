- [x] 2021年1月3日： 学英语的一点经验总结

- Rule 1: 看书不能扣细节 (因为考虑到能效比， BFS 此时总是优于 DFS) 。
- Rule 2: 看书不能记单词 (看不懂的单词跳过，最佳方法是用百词斩粗略过单词)
- Rule 3: 利用“影子阅读”法**一边听音频**， **一边重复复述** 音频内容。 （这叫**“设计模式”**，在写文章或者口语会特别有用，不要求默写，多次阅读就会自然记住了 ）
  - 由于 Rule 3 耗时巨大，必须精选学习材料 (建议 “新概念英语” 和 “雅思听力考题”， 因为新概念是经典， 而雅思听力的场景每一个都是经典)
