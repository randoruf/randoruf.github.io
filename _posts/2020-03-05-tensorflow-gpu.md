## CUDA Capability 

Check whether your graphic card support CUDA. Then **install the latest driver**. 



## Anaconda 

Install anaconda and learn [how to use anaconda](https://bit.ly/tryconda). The [anaconda cheat sheet](https://ran.moe/shared/pdf/conda-cheatsheet.pdf) is also a good starting point.  

After installation, run **Anaconda Prompt** (from start menu). 

To manage environments, use `conda env list` to see how many workspace/environments. You can create a new environment for a new project, and activate this bench `conda activate PROJECT_NAME`. 

If you no longer need this project, `conda env remove --name PROJECT_NAME` to delete it.
