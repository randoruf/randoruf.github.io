建议学习路线

  • C++ Tutorial - Youtube
    • 熟悉 C++ 开发工具
  • 现代 C++ 语言
    • 进入企业后，现代 C++才是真正需要的。
  • CS225 Introduction to Data Structures and Algorithms with C++
    • 数据结构入门
  • 数据结构（C++语言版）初级 - 邓俊辉
    • 看一看就好了，因为 CS225 大部分学过
  • 数据结构（C++语言版）高级 - 邓俊辉
    • 深入了解高级数据结构（红黑树等等）
  • C++ Makefile Tutorial

常用的程序设计方法

  • 过程式
  • 结构化
    • type 类型 , control flow 控制流, function 函数, structure 结构
    • 可以参考 SICP
  • 面向对象
    • encapsulation 封装, inheritance 继承, polymorphic type 多态, generic 类属
