## System Requirement 

Check the **system requirements** of the Windows Terminal before installation, you can look at the **System Requirements** tab. 

![image-20200108135603227](/shared/imgs/image-20200108135603227.png)

Otherwise, you will get this mysterious error (not telling your anything about the requirement.....). 

![image-20200108135548793](/shared/imgs/image-20200108135548793.png)

To upgrade your Windows, please follow the following steps. 

* `Windows` key + `I` to open the **setting** prompt 
* `System` option 
  * ![image-20200108135239072](/shared/imgs/image-20200108135239072.png)
* `About` tab 
  * <img src="/shared/imgs/image-20200108135252009.png" alt="image-20200108135252009" style="zoom:80%;" />
* At `Windows specifications` section, click `Change the product key or upgrade your edition of Windows`. 
  * ![image-20200108135314993](/shared/imgs/image-20200108135314993.png)
* `Windows Update `, and choose the latest version. 
  * ![image-20200108135330354](/shared/imgs/image-20200108135330354.png)

* ![image-20200108135341786](/shared/imgs/image-20200108135341786.png)

## Option 2 : Compile from source code 

Microsoft had released the source code of the Windows Terminal, so you can download from <https://github.com/microsoft/terminal> and then compile by using Microsoft Visual C++. 

See <https://github.com/microsoft/terminal#building-the-code>
