## Matlab and System Resource 

How much system resource is used by Matlab is a metric for knowing the performance of Matlab (and relevant algorithm). 

* Memory used by Matlab: `memory`
* Memory used by variables : `whos`
* Stopwatch : `tic` and `toc`
